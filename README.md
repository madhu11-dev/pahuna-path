# Pahuna Path

**Pahuna Path** is a web-based platform that connects travelers with local communities across Nepal. The main goal is to promote sustainable tourism by helping visitors discover cultural sites, natural attractions, and local accommodations while supporting small businesses such as homestays, teahouses, and guides.

The project focuses on creating a digital bridge between modern travel needs and traditional local experiences, ensuring that tourism benefits reach rural and community-level businesses.

---

## Project Overview

The initial version of Pahuna Path (MVP) is being developed as a 7-week agile project. The MVP aims to include the core functionalities necessary for a functional prototype, focusing on usability, scalability, and local economic impact.

Key focus areas include:

-   User Authentication (Register, Login, Logout)
-   CRUD Management for Places, Accommodations, and Events
-   Route visualization between locations
-   Simple traveler interface for discovering destinations
-   Admin dashboard for managing listings and content

Future versions will include booking features, real-time route optimization, rating and review systems, and community-driven content submissions.
